Brand Guidelines
Colors
(Note: Original guidelines use Pantone for physical production. Below are the filtered core colors with their estimated Hex equivalents for digital UI implementation).

Core Brand Colors (Primary):

B.Duck Yellow: Pantone 116C (approx. #FFCD00) - Primary brand identity, active states, highlights.

B.Duck Orange: Pantone 151C / 1655C (approx. #FF8200 / #FC4C02) - Primary Call-to-Actions (CTAs), prominent icons.

Black: Pantone Black C (approx. #2D2926) - Primary text, heavy contrast elements.

White: WHITE (#FFFFFF) - Main backgrounds and text on dark colors.

Secondary & Theme Colors (Playful/Campaigns):

Theme Magenta/Pink: Pantone 2003C / 212C (approx. #E83668 / #F04E98) - Campaign highlights, playful accents (used heavily in the provided reference).

Sky Blue: Pantone 2995C (approx. #00A9E0) - Informational alerts, secondary tags.

Lime Green: Pantone 375C (approx. #97D700) - Success states, positive KPI badges.

Cool Grays: Pantone Cool Gray 4C & 9C (approx. #B1B3B3 & #75787B) - Subtle borders (border-surface-200), disabled states, meta text.

Typography
Main UI / System Fonts:

Primary Sans-Serif: Poppins (Available weights: SemiBold, Bold, ExtraBold, Black, ExtraBold Italic).
Usage: Used for all structured UI elements, headings, KPI numbers, and data tables to maintain a modern, legible, but slightly rounded/friendly feel.

Display & Theme Fonts (For Graphics/Promotions):

Chunky/Playful: BUBBLEGUM, Alfa Slab One, ALL THE WAY TO THE SUN.

Script/Signature: Thinking Of Betty, Batuphat Script Regular, Aromia Script Black, Avocedit Smoothie Regular, Hunter River Regular, Autograf, Atelas.

Chinese Fonts: 猫啃什锦黑, 华康黑体W12, 华康圆体W9.

Features
Smart Typography Application
Applies Poppins strictly for operational digital UI elements (Dashboards, Tables, Navigation).

Reserves Display and Script fonts (like Bubblegum or Batuphat) ONLY for promotional components (e.g., printed vouchers, marketing modal pop-ups, or the "Referral Celebration Edict").

Automatically falls back to standard sans-serif (Inter/Roboto/Arial) if Poppins is unavailable to preserve system performance.

Color & Styling Application
Translates physical Pantone codes to functional UI Hex codes for web standard adherence.

Distributes B.Duck Yellow and Orange for primary functional elements to anchor the brand.

Uses Secondary/Theme colors (Pink, Blue, Green) for status badges (e.g., Success, Warning, Info) to maintain the "Soft Gamified Enterprise" aesthetic, avoiding harsh default SaaS colors.

Maps Cool Grays to background surfaces and subtle borders to ensure high contrast and readability for data-dense tables.
